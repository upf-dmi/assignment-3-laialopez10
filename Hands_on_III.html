Exercise 1

Build a Random Forest model to classify COVID-19 severity

Using the training cohort provided in Supplementary Table 3, build a Random Forest classification model similar to the one in the publication to distinguish between non-Severe and Severe COVID-19 patients.

Report the following performance metrics from cross-validation:

  • Confusion matrix
  • Accuracy
  • ROC curve and AUC

How well does the Random Forest model separate non-Severe from Severe patients based on proteomic profiles?

# In the paper they used 18 non-severe and 13 severe patients for training data

# Before train the model (because caret convert the class names to column names when it calculates probabilities)
df_model$Severity <- as.character(df_model$Severity)

df_model$Severity[df_model$Severity == "Non-severe"] <- "Non.severe"

df_model$Severity <- factor(df_model$Severity,
                            levels = c("Non.severe", "Severe"))

# Model configuration
set.seed(123)
rf_recipe <- recipe(Severity ~ ., data = df_model)
ctrl <- trainControl(
  method = "cv",
  number = 10,
  classProbs = TRUE, # necessary for ROC
  summaryFunction = twoClassSummary, # calculate ROC, Sens, Spec
  savePredictions = "final"
)

p <- ncol(df_model) - 1 # Max numb of variables to try for RF, exclude classification
mtry_grid <- data.frame( mtry = c(1, 2, floor(sqrt(p)), floor(p/3), p) )

# Train of RF
rf_model <- train(
  rf_recipe,
  data = df_model,
  method = "rf",
  trControl = ctrl,
  tuneGrid = mtry_grid,
  ntree = 1000, #Number of trees built in the research
  importance = TRUE,
  metric = "ROC")


rf_model
plot(rf_model)

# Save the file
png("Data/Processed_Data/rf_model_plot.png", width = 800, height = 600)

invisible(dev.off())

# Extract best CV predictions
best_mtry <- rf_model$bestTune$mtry
predictions <- rf_model$pred
predictions <- predictions[predictions$mtry == best_mtry, ]


# Confusion Matrix
conf_mat <- confusionMatrix(
  predictions$pred,
  predictions$obs,
  positive = "Severe"
)
conf_mat$table


# Accuracy

conf_mat$overall["Accuracy"]

#ROC and AUC

  • The model shows excellent discrimination between severe and non-severe COVID-19 patients based on proteomic profiles, suggesting that proteomic signatures strongly reflect disease severity. We see that the best tune corresponds to the model including 28 proteomic values, this model has a 0.85 Sensibility and Specificity. Is able to identify Severe cases correctly in most of the cases, as it can be observed in the Confusion matrix. The quality of the model gets ratified by a high AUC value.

Exercise 3

External prediction on an independent test cohort

Using the final Random Forest model trained on the full training cohort, predict disease severity for patients in the independent test cohort provided in Supplementary Table 4.

Your task is to:

  • Apply the trained model to the test cohort.
  • Obtain predicted probabilities of Severe disease for each patient.
  • Assign predicted class labels based on these probabilities.
  • Compare predictions to the true clinical labels.

How does the model perform on the independent test cohort?

# Read the test data
test_df <- readxl::read_xlsx("Data/Raw_Data/S4-Independent_Test_Cohort.xlsx", sheet = 2, skip = 1)

# Clean Patient_ID

# Read the metadata
df_test_metadata <- readxl::read_xlsx("Data/Raw_Data/Table-S1.Additional-Demographical-and-Baseline-Characteristics-of-COVID-19-Patients-and-Control-Groups.xlsx", sheet = 2)

df_test_metadata <- df_test_metadata %>%
  rename(
    Patient_ID = `Patient ID a`,
    Group = `Group d`

  mutate(
    Patient_ID = trimws(as.character(Patient_ID)),
    Group = as.numeric(Group)
  ) %>%

  mutate(
  Severity = case_when(
      Group == 2 ~ "Non.severe",

                      levels = c("Non.severe","Severe"))
  )


# Join metadata with proteins

df_test <- df_test[,-1]

#Identify features common with training model
common_features <- intersect(colnames(df_model)[-1], colnames(df_test))

# Subset both datasets to common features
df_model_reduced <- df_model[, c("Severity", common_features)]
df_test_final <- df_test[, c("Severity", common_features)]

# Convert to numeric and impute NA 
df_test_final[, -1] <- lapply(df_test_final[, -1], function(x) as.numeric(as.character(x)))
df_test_final[is.na(df_test_final)] <- 0


# Train the model with reduced features

  • The Random Forest model was initially developed using the full training dataset. For external validation, only the features shared between the training and independent test cohorts were kept. As the test dataset did not contain all the feautres present in the training dataset, a final model was then re-trained on the training data using this reduced feature set to ensure compatibility with the test cohort.
